## Introduction

What is this project?
This project is named "PiCade Matrix MINI" it is a project im making for hackclub fallout. PiCade Matrix MINI is a 
mini arcade cabinet with dual modular slide to remove arcade controllers. it will be using a raspberry pi / orange pi SBC as the
brain of the arcade, rasperry pi pico / esp32s for the controllers (rethinking about bluetooth functionalities).
