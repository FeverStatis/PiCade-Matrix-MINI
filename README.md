# PiCade Matrix MINI

> My project for hackclub fallout

# Introduction

Hello! this is my project for hackclub fallout!!!!! i decided to go with a mini arcade cabinet because
i love arcade games and i have imagined to make this project for almost a year now. 

## What is PiCade Matrix MINI?

PiCade Matrix MINI is a mini arcade cabinet with dual modular slide to remove controllers, it is designed to run classic
arcade games and more

## What is it made of?

The materials for the structure will be made with acryllic sheets / abs sheets. It will use a raspberry pi / orange pi
as the brain of the arcade, it will also use a rasperry pi pico / esp32 for the controllers
